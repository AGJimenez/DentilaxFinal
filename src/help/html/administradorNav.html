Administrador - Navegación

Una vez se logueé con el rol de administrador, encontrará una pantalla en la que podrá acceder a las diferentes fucniones de la aplicación de manera gráfica. Si lo prefiere, también podrá ejecutar combinaciones de teclas para acceder a las mismas de manera más rápida.
